DirText v3.5 Changelog
Export Format Selection (CSV / JSON / TXT)

After clicking the "Export" button, a format selection window pops up

Supports three formats: TXT (tree text), CSV (table), JSON (structured data)

Selected item shows a green solid dot (●), unselected shows a gray hollow dot (○)

After choosing a format, a save dialog appears, and the filename automatically includes a timestamp

Drag-and-Drop Folder

Supports dragging a folder directly into any area of the window

Automatically adds and scans after dropping

Duplicate folders are skipped automatically

Recursive Depth Free Input

The original dropdown is replaced with a "Recursive [N] levels" text input field

Default value is 0 (current level only), -1 means all levels

Input has a 1-second debounce; scanning is triggered automatically after typing stops

Usage instructions have been added to the welcome page
